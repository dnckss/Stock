  엔드포인트                                                                                                                                                                                       
   
  GET /api/strategy (변경 없음)

  {                                                                                                                                                                                                
    // ── 기존 필드 (유지) ──                               
    "market_summary": "시장 전체 상황 요약 (뉴스·경제·기술적 지표 통합 분석)",

    "sector_chart": [...],

  신규 필드 처리 가이드                                                                                                                                                                            
                                                            
  ┌──────────────────────────┬───────────────┬─────────────────────────────────────┬───────────────────────┐                                                                                       
  │           필드           │     타입      │            null/빈 가능             │         설명          │
  ├──────────────────────────┼───────────────┼─────────────────────────────────────┼───────────────────────┤                                                                                       
  │ top_picks[].direction    │ string        │ "BUY" / "SELL" / "HOLD"             │ 매매 방향             │
  ├──────────────────────────┼───────────────┼─────────────────────────────────────┼───────────────────────┤                                                                                       
  │ top_picks[].confidence   │ string        │ "high" / "medium" / "low"           │ AI 확신도             │                                                                                       
  ├──────────────────────────┼───────────────┼─────────────────────────────────────┼───────────────────────┤                                                                                       
  │ news_themes              │ array         │ [] (빈 배열 가능)                   │ 뉴스 핵심 테마 목록   │                                                                                       
  ├──────────────────────────┼───────────────┼─────────────────────────────────────┼───────────────────────┤                                                                                       
  │ news_themes[].sentiment  │ string        │ "positive" / "negative" / "neutral" │ 테마 감성             │
  ├──────────────────────────┼───────────────┼─────────────────────────────────────┼───────────────────────┤                                                                                       
  │ econ_impact              │ string | null │ null 가능                           │ 경제 이벤트 분석 요약 │
  ├──────────────────────────┼───────────────┼─────────────────────────────────────┼───────────────────────┤                                                                                       
  │ risk_events              │ array         │ [] (빈 배열 가능)                   │ 리스크 이벤트 목록    │
  ├──────────────────────────┼───────────────┼─────────────────────────────────────┼───────────────────────┤                                                                                       
  │ risk_events[].risk_level │ string        │ "high" / "medium" / "low"           │ 리스크 수준           │
  └──────────────────────────┴───────────────┴─────────────────────────────────────┴───────────────────────┘                                                                                       
                                                            
  주의사항                                                                                                                                                                                         
                                                            
  - news_themes, econ_impact, risk_events는 LLM이 생략할 수 있으므로 없거나 null/빈 배열일 때 안전하게 처리 필요                                                                                   
  - top_picks의 기존 rationale 필드는 유지되며, direction과 confidence가 추가됨
  - OpenAI 실패 시 fallback 응답에서도 동일 구조가 유지됨 (빈 값으로)
